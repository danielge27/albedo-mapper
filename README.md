# Urban Albedo Mapper

Analyze surface reflectivity and urban heat risk using satellite imagery.

## Features
- Enter any address to load satellite imagery
- Interactive map with zoom/pan controls
- Automatic surface detection (roofs, roads, vegetation, etc.)
- Albedo and heat index calculation
- Policy recommendations for urban cooling
- Downloadable PNG report

## Local Development

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/albedo-mapper.git
cd albedo-mapper

# Install dependencies
pip install -r requirements.txt

# Run locally
python app.py
```

Open http://localhost:5000

## Deployment

This app is configured for deployment on Render.com (free tier).
